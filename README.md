## Getting Started
How to run code.

About 1D CNN.
1.	run ./1D-CNN/examples/raw_data_augument.py to prepare the training dataset and testing dataset;
2.	run ./1D-CNN/train.py to train CNN model;
3.	run ./1D-CNN/test.py to test the trained model. 

About 2D CNN.
1.	run ./2D-CNN/preprocessing/preprocessingt.py to prepare the training dataset and testing dataset;
2.	run ./2D-CNN/train.py to train CNN model;
3.	run ./2D-CNN/test.py to test the trained model. 

About 3D CNN.
1.	run ./3D-CNN/preprocessing/augmentation_processingt.py to prepare the training dataset and testing dataset;
2.	run ./3D-CNN/train.py to train CNN model;
3.	run ./3D-CNN/test.py to test the trained model.
